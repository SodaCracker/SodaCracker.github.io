Project B

Zlongame · Blackjack Studio — 2019–Present

A large-scale JRPG with real-time combat, expansive exploration, and rich, layered environments.

Key Contributions
  • Combat UI & timing accuracy: built the core battle UI and full-screen damage-number system — object-pooled by design to avoid native-API GC. The hard part was timing: keeping numbers perfectly in sync with damage resolution through repeated combat refactors and changing damage mechanics.
  • NavMesh customization: customized a NavMesh generator from an open-source base and built a dedicated pathfinding plugin for complex sloped terrain, simplifying node configuration for large open-world maps.
  • Custom tooling: built "UI Layer Manager" and "Footstep Layer Editor" to streamline art and design workflows.
  • Audio immersion: developed a footstep-sound blending system that handles transitions across complex surface materials.
